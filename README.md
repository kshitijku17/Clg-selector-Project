# 🎓 College Selector

## Overview

College Selector is an AI-powered web application that helps students discover colleges that best match their preferences. It uses a custom scoring algorithm and AI-generated insights to provide personalized college recommendations.

## Features

-  Location-based filtering
-  Budget-based recommendations (Tuition Fees + Living Expenses)
-  Average Placement filtering
-  Top 3 personalized college recommendations
-  AI-powered college insights
-  Official college website links
-  Custom scoring algorithm

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### AI
- Groq API (OpenAI OSS 120B)

### Dataset
- Custom CSV Dataset

## How It Works

1. Select your preferred location.
2. Select your budget.
3. Select your expected average placement.
4. The system scores every college based on your preferences.
5. The Top 3 most suitable colleges are recommended.
6. AI generates a brief overview for each recommended college.
